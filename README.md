Food App UI/UX Design
Project Overview

This project showcases a **Restaurant Menu & Ordering Food Application UI/UX Design**, created as part of a design task provided by **Future Interns**. The goal of the project was to design a clean, modern, and user-friendly food ordering experience where users can browse menu categories, view food items, add items to the cart, and place orders easily.

The design focuses on **clarity, visual appeal, and smooth user flow**, inspired by real-world food delivery and in-restaurant ordering applications.
Project Objectives

* Design an intuitive digital food menu interface
* Create category-based food browsing (Starters, Mains, Desserts, Drinks, etc.)
* Ensure a smooth ordering and checkout flow
* Apply modern UI principles suitable for mobile screens
* Build a portfolio-ready, real-world design case
Screens Designed

* Home / Restaurant Overview Screen
* Menu Categories & Food Listing Screen
* Food Item Detail Screen
* Cart & Order Summary Screen
* Checkout Screen
